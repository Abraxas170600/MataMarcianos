README - [Retro Crysis]

Instrucciones para Ejecutar el Proyecto:
1. Clonar el Repositorio
Abre el siguiente link de Github para clonar el repositorio utilizando la herramienta de libre elección.

2. Abrir el Proyecto en Unity
Abre Unity Hub.
Haz clic en Add y selecciona la carpeta del proyecto clonado.
Asegúrate de que estás usando la versión correcta de Unity (2022.3.47f1).
Abre el proyecto desde Unity Hub.

3. Entrar a la Escena Inicial
En el editor de Unity, navega al directorio Assets/Scenes.
Abre la escena principal (MainMenu.unity) haciendo doble clic en ella.
